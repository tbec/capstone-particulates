Analytics

The analytics view is where you can run queries against selected devices. Upon return, you can get a preview of that device with the data display feature. Satisfactory data sets can be downloaded as a csv by toggling the tool mode to download.

Set Management

The analytics view consists of two sets: The All Devices set and query Set. The All Devices are populated from devices chose through the map view while the Query Set are the current devices you'd like to query against.

Pollution Previewer

The Pollution Previewer is a two in one tool. After setting the query parameters, you have the option of either previewing the data presented or downloading the dataset. Hit the toggle button to see the query options change between 'preview' and 'download'.

Toggle Preview/Download

Set Query Parameters

Table Navigation

After setting the query parameters, hit the preview button. A table will be constructed based off of the Query Set devices and parameters chosen. Each row previews the most recent hour of data in the selected time window. To see the dataset just click on a row. To show a graphical view hit the graph icon. This allows you to view the result set in either a circle or bar graph. Bar graph views are combined into 5 minute averages the the 60 minute preview.
